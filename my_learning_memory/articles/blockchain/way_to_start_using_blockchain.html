由 0 開始用 Ethereum
如何開始

要使用銀行戶口，首先就要去銀行開戶口，用Ethereum都差不多，但基於其去中心化的特性，你不需要個人資料就可以建立戶口（account）。

戶口的建立是由錢包軟件（wallet）管理，而目前最常用的wallet就是Metamask。

當你下載了wallet，它就會為你建立account，而account即是私鑰（private key）和地址（address）的組合。但不用擔心，你的wallet會為你管理。

下一步，就要嘗試取得一些ETH，取得的方法有很多，方法不外乎：從找換店或交易所購買都可以。買的時候要提供address，對方就會把ETH發送給你。

接下來，我們嘗試把剛才收到的ETH發送到其他地址。你可以去 CoinBase 開戶口，它是一個網上錢包，你可以在那裏取得另一個ETH asdress，然後發送進去。

就這樣，你就完成了開戶、以及購買、接收和發送ETH的步驟。
