由0開始用Ethereum
基本篇

要使用銀行戶口，首先就要去銀行開戶口，用Ethereum都差不多，但基於其去中心化的特性，你不需要個人資料就可以建立戶口（account）。

戶口的建立是由錢包軟件（wallet）管理，而目前最常用的wallet就是Metamask。

當你下載了wallet，它就會為你建立account，而account即是私匙（private key）和地址（address）的組合。但不用擔心，你的wallet會為你管理。

下1步，就要嘗試取得1些ETH，取得的方法有很多，方法不外乎：由找換店或交易所買、或者挖礦都可以。買的時候要提供Address，對方就會把ETH發送給你。

接下來，我們嘗試把剛才收到的ETH發送到其他地址。你可以去 CoinBase 開戶口，它是1個網上錢包，你可以在那裏取得另一個ETH asdress，然後發送進去。

就這樣，你就完成了開戶、以及購買、接收和發送ETH的步驟。

如何防範加密貨幣詐騙

加密貨幣近年來備受關注，為投資和金融創新提供了令人興奮的機會。然而，隨著其流行，詐騙行為也隨之增加，目標鎖定在不知情的人士身上。本文旨在協助您識別並避免常見的加密貨幣詐騙，確保您的投資安全。

了解常見的加密貨幣詐騙
  1. 虛假交易所：騙子會搭建仿冒真實交易所網站，誘使用戶存入資金後被盜取。
  2. 網路釣魚攻擊：這些攻擊透過欺騙性郵件或訊息來竊取私鑰或登錄詳細資訊。
  3. 愛情詐騙：騙子在網上建立信任關係，之後要求進行投資或直接轉移加密貨幣。
  4. 龐氏騙局：這種騙局承諾高回報，利用新投資者的資金來支付先前的投資者，當他們無法維持支付時就會崩潰。
防止加密貨幣詐騙的方法
  1. 驗證交易所：確保使用的交易所是合法的，通過可信任來源進行交叉核對，並檢查URL的準確性。
  2. 啟用雙重認證 (2FA)：在密碼之外增加一層安全性保護。
  3. 保護私鑰：永遠不要分享您的私鑰，因為它們授予資金訪問權限。
  4. 投資前調查：在進行任何投資之前，徹底研究項目，檢查白皮書和團隊的可信度。對過高回報保持警惕。
  5. 使用硬體錢包：這種脫機設備提供比線上錢包更高的安全性。
  6. 定期軟件更新：保持軟件更新以防止漏洞被利用。
如果受騙該怎麼做

如果您懷疑遇到詐騙，請立即向地方當局和相關交易所報告。諮詢區塊鏈專家可能會幫助追蹤交易，儘管恢復資金通常困難重重。

結論

加密貨幣詐騙帶來重大風險，但透過覺醒和警惕可以緩解這些威脅。通過遵循上述預防措施並保持資訊更新，您可以在加密貨幣領域安全航行。始終將謹慎和盡職調查放在首位，以保護您的投資和個人資訊。
